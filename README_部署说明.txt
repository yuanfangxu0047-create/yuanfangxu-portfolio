部署建议：

1. 将当前文件夹整体上传到 Netlify。
2. 部署完成后，请将站点名改为：yuanfangxu-portfolio
3. 改名后，个人网页链接将为：
   https://yuanfangxu-portfolio.netlify.app/
4. 二维码入口链接将为：
   https://yuanfangxu-portfolio.netlify.app/qr

说明：
- 当前网页中的复制链接按钮与二维码，均已按上述域名进行配置。
- 若你在 Netlify 中使用了其他站点名，请同步修改 index.html 与 qr/index.html 中的 PUBLIC_URL。
